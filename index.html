Web Engineering

"Disciplined, systematic application of software engineering principles to the development, deployment, and maintenance of high-quality web-based systems"

Core components of Web Engineering

  • Frontend

  • Backend

  • Database

1. Frontend:

"The part of a website or application that users directly interact with, focusing on design, layout, and user experience."

It includes

  • HTML
  • CSS
  • JS

2. Backend:

"The server-side logic that processes requests, manages data, and connects the frontend with the database."

We can use

  • PHP
  • PYTHON
  • NODE JS
  • EXPRESS JS

3. Database:

"A structured system used to store, manage, and retrieve data efficiently for web applications."

  1. SQL
    • MY SQL
    • POSTGRE SQL
    • ORACLE
  2. NO SQL
    • MONGO DB
    • REDIS
    • CASSANDRA
